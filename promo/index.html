자동으로 찾아주는 사람들

같은 WiFi(서브넷)의 사용자를 실시간으로 발견하고, 응답 없는 사람은 목록에서 자동으로 정리합니다.

전체 채팅 & 귓속말(DM)

이름을 탭해 상대를 선택하면 그 사람에게만 가는 🔒 귓속말. 선택을 풀면 다시 전체 대화로.

서버 없는 진짜 P2P

메시지는 어떤 서버도 거치지 않고 기기끼리 직접 오갑니다. 세션이 끝나면 기기 메모리에서 사라져요.

신고 · 차단

불편한 사용자는 길게 눌러 바로 신고하거나 차단. 차단하면 해당 사용자의 메시지가 더 이상 보이지 않습니다.

가입 없이 1초 시작

계정도 비밀번호도 없습니다. “오늘의 나는 누구?”에 이름만 정하면 즉시 참여.

광고 · 추적 제로

분석 도구, 광고, 제3자 SDK가 전혀 없습니다. 위치 등 민감 권한도 사용하지 않아요.

🪟 Windows

데스크톱에선 한 단계 더

안드로이드의 모든 채팅 기능에 더해, 윈도우 앱에는 완전 오프라인 AI 어시스턴트가 내장됩니다

채팅 기능 그대로

자동 발견, 전체 채팅, 🔒 귓속말, 신고·차단까지 모바일과 동일하게 큰 화면에서.

방화벽 허용 한 번이면 끝

별도 설정 없이 동작합니다. 첫 실행 시 방화벽 허용만 누르면 바로 같은 네트워크에 연결돼요.

안정적인 연결 유지

최초 발견 이후엔 유니캐스트로 직접 연결을 유지해, 공유기가 멀티캐스트를 끊어도 대화가 이어집니다.

실제 캡처

메뉴판만 넣으면, 우리 가게 AI 완성

AI 이름·프롬프트를 정하고, 텍스트·이미지·문서로 메뉴를 학습시키면 — 손님 질문에 바로 답하는 챗봇이 됩니다. 모두 내 PC 안에서.

메뉴판 학습 (텍스트·이미지·문서)

메뉴와 가격을 직접 입력하거나, 메뉴판 사진을 OCR로 읽고, PDF·텍스트 문서를 그대로 학습시킬 수 있어요.

메뉴에 대해 무엇이든 질문

“매운 거 빼고 추천해줘”, “2만원 이하 인기 메뉴는?” — 학습한 메뉴를 바탕으로 자연스럽게 답합니다.

내 PC 안에서만 처리

질문도 메뉴 데이터도 외부로 나가지 않습니다. 네트워크가 끊겨도 AI는 그대로 작동해요.

이름·성격·프롬프트 커스터마이즈

AI 이름과 역할 프롬프트를 자유롭게 바꾸고, 컨텍스트 사용량까지 한눈에 확인할 수 있습니다.
